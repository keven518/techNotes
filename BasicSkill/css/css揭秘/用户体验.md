* 遮罩层

以下这种方案比较稳妥。可以在遮罩层监听相应的事件。

```css
.overlay {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: rgba(0,0,0,.8);
}

.show {
  position: absolute;
  z-index: 1;
}
```
